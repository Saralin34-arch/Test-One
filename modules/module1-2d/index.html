This module explores interactive canvas experiments with organic shapes, color theory, and motion design principles. Using p5.js, we create dynamic visual experiences that demonstrate fundamental computational design concepts.

Interactive Experiments

Three core experiments demonstrating different aspects of 2D computational design

01

Organic Grid Generation

An interactive grid system that responds to mouse movement, creating organic patterns through algorithmic design. This demonstrates how simple mathematical principles can create complex visual forms.

02

Bouncing Ball Physics

A physics-based animation demonstrating motion, collision detection, and gravity simulation. The ball responds to user interaction and environmental boundaries.

03

Zoom & Pan Controls

Interactive navigation system allowing users to explore large-scale visual compositions through zoom and pan operations. This demonstrates spatial navigation in digital environments.
