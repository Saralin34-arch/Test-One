Module 5

Geospatial Structures

Interactive mapping using Mapbox GL JS, demonstrating markers, line features, and polygon regions

Student Sara Lin • Instructor Catherine Griffiths

This section explores geospatial data visualization using Mapbox GL JS, demonstrating how to create interactive maps with markers, line features, and polygon regions. These examples show how to represent different types of spatial data and create engaging user experiences.

Mapbox Experiments

Three core experiments demonstrating different aspects of geospatial visualization

02

Mapbox Advanced: Line Features and Regions

An advanced Mapbox example demonstrating line features and polygon regions using GeoJSON data. This map shows Manhattan with interactive line features and polygon regions. Features include hover effects, click interactions, and toggle controls.

03

Mapbox External Data: Loading GeoJSON from Files

A demonstration of loading and visualizing data from external GeoJSON files, showing how to separate data from visualization logic. This example loads NYC neighborhood data and automatically frames the map to fit all the data.
