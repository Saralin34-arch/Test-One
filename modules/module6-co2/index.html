Environmental Data Visualization

This interactive visualization presents CO2 emissions data across all U.S. states using a bubble map format. Each bubble represents a state, with size indicating total emissions and color intensity showing the percentage of emissions from petroleum sources. The visualization includes interactive features such as hover effects, tooltips with detailed information, and a legend explaining the data encoding.

Key Features

  • Bubble Size: Represents total CO2 emissions in million metric tons
  • Color Intensity: Shows percentage of emissions from petroleum sources
  • Interactive Tooltips: Display detailed state information on hover
  • Geographic Accuracy: States positioned according to their actual geographic coordinates
  • Data Source: U.S. Energy Information Administration (EIA)

Technical Implementation

  • D3.js: Core visualization library for data binding and DOM manipulation
  • TopoJSON: Geographic data format for efficient map rendering
  • SVG Graphics: Scalable vector graphics for crisp visualization
  • Responsive Design: Adapts to different screen sizes
