Exhibition Overview

This collection explores the intersection of Bauhaus principles and computational design through interactive 2D canvas experiments. Each demo demonstrates different aspects of geometric abstraction, color theory, and user interaction.

Exhibition Items
01

2D Drawing Demo

Organic grid composition with colorful, soft shapes and geometric elements. Features peanut, clover, arch, cloud, diamond, and circle shapes arranged in a dynamic composition.

Interactive Geometric Color Theory
2D Drawing Demo
02

Bouncing Ball Demo

Dynamic animation featuring two balls with different colors and trajectories. Demonstrates motion design principles and physics-based interactions.

Animation Motion Physics

Zoom & Pan Demo

Interactive canvas with zoom and pan functionality. Allows users to explore detailed compositions through spatial navigation.
